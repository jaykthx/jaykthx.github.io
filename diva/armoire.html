Armoire is a customisation editor for Project DIVA Mega Mix+ and other supported titles.

Description Armoire is a tool made in C#, using WPF for the user interface.
To use it, please ensure you have one of the supported titles and the following requirements:
  • A 64-bit CPU
  • .NET Framework 4.8 Runtime
